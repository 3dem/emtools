=======
emtools
=======

Basic utilities for CryoEM data manipulation. It is available as a Python module (https://pypi.org/project/emtools).

Installation
------------

``pip install emtools``

Or for development:

.. code-block:: bash

    # Or for development
    git clone git@github.com:3dem/emtools.git
    cd emtools
    pip install -e .
